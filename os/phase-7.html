form-data CVE and Revenue Tracking were documented — not unknowns
Both came through phase dashboards and ClickUp. Pulled from source rather than treating as ambiguous.
OS nav rework requires a full conversation before any code
Not just a menu rename — involves page content, structure, and new section definitions. Debate-first, code-second.
SSE is not reliable for long-running operations on Railway — use polling
Railway's nginx proxy has its own connection timeout independent of SSE heartbeats. Pattern established: POST fires the job and returns immediately; frontend polls a status endpoint every 2s. Applied to GEO audit; default pattern going forward.
Python on Railway requires Dockerfile; Dockerfile breaks Railway's delta snapshot
Port any Python scripts to Node.js rather than fight the build system. geo_audit.py → geo-audit.js is the precedent.
GEO audit history goes in Google Sheets, not the container filesystem
Railway containers are ephemeral — filesystem resets on every redeploy. Google Sheets: durable, Drive already wired, native charting.
